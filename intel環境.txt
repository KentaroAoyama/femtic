intel/oneapiのイメージをpull
	docker pull intel/oneapi

コンテナ起動
	初回：docker run --volume :/home/:rw -it --name=intel-oneapi  intel/oneapi
	二回目以降：docker exec -it intel-oneapi bash
	(debug用)：docker run --volume $PWD:/home/:rw -it --cap-add=SYS_PTRACE --security-opt="seccomp=unconfined" --name=femtic-dev femtic-dev
コンテナ作成 & コンテナ内でg++をinstall (TOOD: Dockerfileでg++のinstallまで完了させる or shell書く)
	apt-get update
 	apt-get install -y sudo
	sudo apt update
	sudo apt-get install g++
	sudo apt-get install intel-basekit
	sudo apt-get install intel-hpckit
	sudo apt-get install gfortran
	sudo apt-get install libomp-dev
	sudo apt-get install gcc-multilib
	sudo apt-get -y install openmpi-bin
	sudo apt-get install libblacs-mpi-dev
